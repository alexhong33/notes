# Python3 元组

### 与list的区别 1
 元组与列表类似, 但元组的元素是不能修改的
>元组中的元素值是不允许删除的, 但我们可以通过del语句来删除整个元组

>t = (1, 6, 3)  
>del tup  
>print tup 报异常 不存在t


### 与list的区别 2
* 空元组 t = ();
* 元组中只包含一个元素时, 需要在元素后面添加逗号 t = (50, )


###内置函数

* len(tuple) 计算元素个数
* max(tuple) 返回元祖中元素最大值
* min(tuple) 返回元组中元素最小值
* tuple(seq) 将列表转换为元组
